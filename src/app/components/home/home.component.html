Welcome to E-shopping Project!

Developer: Suresh Prajapati

This is demo project implementing Spring Boot Microservices, running on google cloud. Angular is also deployed on google Cloud Platform. It took around 2 weeks for complete full stack development. (Stay Home Stay Safe)

Features: (BackEnd)

  • Restfull Service running on cloud based on Java.
  • JWT Authentication for Role based authentication.
  • Email Feature for verifying email address (6 digit code verification). This is required if customer wants to rate, comment or reply on specific product. If you login with oath2 i.e. Login with google no verification required.
  • Mongodb for Storing information. (Auto Increment of PrimaryKey implemented manually)
  • Search Feature (requires at least 3 characters) matches word that is either in name of product or description on product. (Implemented using custom Query)
  • Admin can add products and vendors.
  • Password encryped using bycrypt.
  • Search By Category will be added soon...

Features: (FrontEnd)

  • Angular 8 with Bootstrap 4
  • Login with google feature Oath2.
  • Implemented Http Interceptors, Custom Pipes, Guard.
  • Implemented component to component communication using service. (cart subscribes to items in cart)
  • Used debounce time so that we hit server only once when user clicks multiple times + icon in cart items
  • Get method cached on client side using service for improving perfomance.
